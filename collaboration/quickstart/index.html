Quickstart

เราลองมาทำแบบฝึกหัดเล็ก ๆ ดูตั้งแต่ต้นจนจบ ว่าถ้าอยากแก้เนื้อหาในหน้าประวัติของตัวเองในเว็บไซต์หลักของ PIER จะต้องทำยังไง

  1. ไปที่ pier-source ซึ่งเป็น repository ของเว็บไซต์หลัก
  2. เข้าไปที่ content/members/firstname-lastname (เปลี่ยน firstname-lastname เป็นชื่อของคุณเอง)
    ใน folder นั้น ควรจะมีอยู่ 3 files ด้วยกัน คือ
    • index.th.md เก็บประวัติสำหรับหน้าภาษาไทย
    • index.en.md เก็บประวัติสำหรับหน้าภาษาอังกฤษ
    • ไฟล์รูปภาพ
  3. เลือกไฟล์ที่ต้องการแก้ไข สมมติว่าต้องการแก้ไขประวัติภาษาไทย ก็เลือก index.th.md
  4. กดปุ่มรูปดินสอ (Fork this project and edit the file หรือถ้าเคย fork แล้วก็จะขึ้นว่า Edit the file in your fork of this project)
  5. แก้เนื้อหาที่ต้องการ จากนั้นกดปุ่ม Propose changes สีเขียวด้านล่าง
    ไอเดีย

    ไหน ๆ ก็สร้าง GitHub account แล้ว จะลองเพิ่มเข้าไปใน profile เลยก็ได้นะ สามารถใส่ไว้ตรงไหนก็ได้ก่อน --- (เรียกว่าส่วน frontmatter)

    /content/members/firstname-lastname/index.th.md
    ---
    ...
    email: youremail@domain.com
    github: github-username
    ...
    ---
  6. จะมีหน้าจอขึ้นมาเปรียบเทียบว่ามีเนื้อหาอะไรเปลี่ยนไปบ้าง เมื่อเรียบร้อยแล้ว กด Create pull request เพื่อขอให้ admin นำสิ่งที่เปลี่ยนแปลงนี้ไปรวมเข้ากับเนื้อหาหลัก
  7. ตั้งชื่อ pull request (PR) ให้สื่อความหมาย เช่น "Edit Firstname's profile" แล้วกด Create pull request
  8. ถ้าเข้าไปดูใน list ของ PR จะเห็น PR ใหม่ถูกสร้างขึ้นตามรูปด้านล่าง
  9. หลังจาก admin ตรวจสอบและ merge PR แล้ว จะเห็นความเปลี่ยนแปลงหลังการ build ครั้งต่อไป

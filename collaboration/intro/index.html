ร่วมกันปรับเนื้อหาเว็บไซต์

เนื้อหาทั้งหมดของทั้งเว็บไซต์ PIER✗ และเว็บไซต์หลักของ PIER ถูกเก็บอยู่ใน GitHub ในรูปของไฟล์ Markdown ซึ่งเปิดให้บุคลากรของ PIER เห็นได้เท่านั้น guide นี้จะแนะนำวิธีปรับแก้เนื้อหาในเว็บคร่าว ๆ ให้

info

คุณต้องเป็นบุคลากรของ PIER ถึงจะมีสิทธิ์ปรับแก้เนื้อหาในเว็บไซต์ได้ ถ้าคุณต้องการแก้ไขเนื้อหาแต่ยังไม่ได้เป็นบุคลากรของ PIER ลองมาร่วมงานกับเราดู

ร่วมแก๊งค์ PIER✗​

  1. ก่อนอื่นเราต้องมาสร้าง account ก่อน (ถ้ามี account แล้วสามารถข้ามขั้นตอนนี้ได้เลย)

    • ไปที่ https://github.com/ แล้วกด Sign Up
    • ทำตามขั้นตอนที่ปรากฏ

  2. หลังจากมี GitHub account แล้ว เข้าไปที่หน้าทิชชู่ เอ๊ย Issue เพื่อขอ invite เข้า PIER✗

    note

    เมื่อได้รับ invite แล้วจะมี email ส่งไปแจ้ง หรือสามารถเข้าไปที่ PIER✗ แล้วกด Accept Invite ในแถบด้านบนได้เลย

โครงสร้างของ PIER✗​

ใน PIER✗ ซึ่งเป็น GitHub organization จะมี repository ที่เกี่ยวกับเว็บไซต์อยู่สองอัน คือ

  • pier-source เป็น source code ของเว็บไซต์หลัก
  • pier-x เป็น source code ของเว็บไซต์นี้

ทั้งสองเว็บไซต์มีโครงสร้างคล้าย ๆ กัน คือเนื้อหาทั้งหมดจะถูกเก็บอยู่ใน folder ที่ชื่อ content ยกตัวอย่างเช่น ใน pier-source/content ก็จะมี subfolder ที่เก็บเนื้อหาต่าง ๆ ของเว็บไซต์หลักไว้ ไม่ว่าจะเป็น abridged, dp, members ฯลฯ ถ้าต้องการแก้ไขเนื้อหาที่เกี่ยวกับส่วนไหน ก็เข้าไปแก้ใน folder นั้น ๆ ได้เลย

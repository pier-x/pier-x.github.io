PIER Style Guide

สถาบันวิจัยเศรษฐกิจป๋วย อึ๊งภากรณ์ เผยแพร่งานเขียนจากนักวิชาการหลายคน เพื่อให้งานเขียนที่ออกสู่สาธารณชนมีความเป็นอันหนึ่งอันเดียวกัน จึงจำเป็นต้องมี style guide นี้ขึ้น

คำแนะนำทั่วไป​

การเขียนบทความหรืองานวิจัยควรคำนึงถีงผู้อ่านเป็นหลัก ซึ่งกลุ่มผู้อ่านก็จะแตกต่างไปตามผลิตภัณฑ์หรือประเภทของงานเขียนนั้น ๆ งานเขียนบางประเภท เช่น Discussion Paper หรือ PIERspectives เน้นกลุ่มผู้อ่านที่เป็นนักเศรษฐศาสตร์หรือนักวิชาการด้วยกัน ในขณะที่งานเขียนอีกประเภท เช่น aBRIDGEd, เศรษฐศาสตร์เข้า "ท่า", PIER Blog หรือการสรุปการบรรยายต่าง ๆ มุ่งเน้นสื่อสารให้กลุ่มประชาชนทั่วไป

เนื่องจากงานเขียนเชิงวิชาการจะค่อนข้างยืดหยุ่น คำแนะนำในส่วนนี้ จะเน้นไปที่งานเขียนสำหรับประชาชนทั่วไปเป็นหลัก

  • ควรใช้คำอธิบายที่ไม่ยากเกินไปนัก หรือหากจำเป็น ก็ควรมีส่วนที่อธิบายเนื้อหายาก ๆ ในลักษณะที่คนทั่วไปเข้าใจได้
  • ไม่ควรแสดงผลออกมาในรูป regression table ที่คนทั่วไปเข้าใจได้ยาก แต่สรุปออกมาเป็นคำพูดที่เข้าใจได้ง่ายกว่า
  • ถ้างานเขียนเป็นการสรุปงานเชิงวิชาการชิ้นอื่น ควรมีอ้างอิงไปยังบทความฉบับเต็มด้วย เช่น

    บทความนี้สรุปจากงานวิจัยฉบับเต็ม เรื่อง …

  • แต่ละย่อหน้าควรมีใจความสำคัญเพียงใจความเดียว และหากใจความสำคัญนั้นอยู่ในประโยคแรกสุด ก็จะทำให้ผู้อ่านอ่านผ่าน ๆ โดยที่ยังจับประเด็นได้
  • ย่อหน้าไม่ควรจะยาวเกินไป
